# APDS (Attack Prevention & Detection System)

Full-stack demo:
- Backend: Spring Boot (Java 17) on `http://localhost:8080`
- Frontend: React (Vite) on `http://localhost:5173`

## Run

### Run both
- On Windows: run `run_all.bat`

### Run backend only
- On Windows: run `run_backend.bat`

### Run frontend only
- On Windows: run `run_frontend.bat`

## URLs
- Dashboard: `http://localhost:5173`
- Backend health/demo endpoint: `http://localhost:8080/hello`
